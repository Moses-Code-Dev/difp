# DIFP Specification Documents

This directory contains the official specification documents for the Djowda Interconnected Food Protocol (DIFP).

## Files

- `DIFP-v0_4.docx`: The full specification in Microsoft Word format. You can download the latest version from [djowda.com/assets/docs/DIFP-v0_4.docx](https://djowda.com/assets/docs/DIFP-v0_4.docx).
- [SPEC.md](../SPEC.md): The full specification in Markdown format, optimized for GitHub viewing.

## Note on previous versions

Earlier versions (v0.1–v0.3) have been superseded by v0.4. For historical reference, see the git history.
